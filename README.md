## Nuevas Tendencias de Autos Electricos

Guía comparativa de autos eléctricos  de distintos países del mundo, 

Que incluye:

-	SUVs eléctricos 
-	Sedanes eléctricos
-	Híbridos enchufables

Cada modelo  contiene su precio, modelo, marca, año y colores de que vienen mostrando:

-	Precio en dólares
-	Modelo según se especifique
-	Año del cual le interesaría comprar
-	Marca que desearía

Se aplicaron las siguientes propiedades:

- Se diseñaron 2 layouts diferentes (Mobile y Escritorio)
- Se  considera que la versión de escritorio debe activarse sobre los 576px. 
- Se utiliza CSS Grid y su propiedad grid-template-areas .  
- Se utiliza  Flexbox para distribuir los elementos dentro de cada sección de los layouts.

 La página web cuenta con  4 secciones: 
-	Menú de navegación ,
- Sección de filtros de búsqueda, 
- Sección Principal,
-	Pie de página
